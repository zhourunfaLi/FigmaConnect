figma museumCAT 微信小程序设计说明

项目介绍：这个项目是展示世界伟大艺术的微信小程序，可以自适应不同设备排版，核心功能是这些艺术作品都是超高清原图，由于艺术图像文件非常大所以通过krpano技术实现超大文件分层加载，用户通过拖拽滑块或双指缩放就能观看到作品的细节，让用户获得如同博物馆参观原作的用户体验。还有介绍文字，讲解视频等。用户可以下载原始文件，互动评论等。
项目分为收费和非收费会员，以及部分内容需要用户观看广告才能继续体验。下面我将各网页简单的介绍主要功能

1、latest page
	是小程序进入的首页
	1.1 weixinbar，是本项目在微信部署后的自带微信顶端工具条。
	1.2 header item，作品的分类标签，我在figma中只做了3个页面参考分别是latest page，hotest page和cityp page，标签页可以编辑。
	1.3 works list section	是对应分类标签的作品列表，由后台添加，由于作品长宽尺寸不同，排版参考了https://www.pinterest.com首页的瀑布式作品布局，“抱银貂发的女子”是模拟作品名称。SVIP标识说明此作品是SVIP特权。在瀑布式作品列表中可以添加广告，要求广告样式接近作品图标，并用小字标注是广告。在上传作品封面图时可以调整封面图显示区域和长宽。
	1.4 footer-home&user
	是底部栏可以返回到首页和用户页面
	
2、login page
	1.1新用户在点击任何作品时，需要授权微信登陆（可以扩展成FB或者google账号登陆），并且完善注册信息验证如手机验证码或邮箱验证码。
	
3、workdetails page
	3.1 works show 是嵌入由krpano展示的艺术作品，作品分为免费版本和SVIP版本
	3.2 slider bar是控制作品放大缩小的滑块，
	3.3 wokrs title 作品标题也是作品分享后显示的标题文字
	3.4 X Times 是显示当前放大的倍数，这个倍数是图片最大尺寸与当前设备stage分辨率的比较。
	3.5 works video 插入作品的介绍视频
	3.6 works info 介绍作品的文字，可以是滚动页面，介绍文字中有些知识点可以跳转的超链接。
	3.7 FAQ 是相关艺术作品的趣味问答题，从N个题目中随机抽取5个显示。用户完成答题后，点击提交验证答案需要观看广告后再能知道对错。答案页请设计。
	3.8 Comment 用户评论，观看更多评论需要点击广告后才能看到
	3.9 dowload 作品下载，需要SVIP会员实现作品下载。

4 PRICE PAGE
	4.1 Membership Fees 三种不同会员价格价格，由后台定义点击后跳转至微信只支付
	4.2 Invite Code 用户输入邀请可以优惠5元，价格可自定义。
	4.3 SVIP Rights Introduction
	SVIP用户特权限功能：A.所有作品观看;B.去除广告；C.高清下载作品；D.作品全屏观看
	
5 NotVip PAGE
	如果不是SVIP会员，弹出此页。类似PRICE PAGE
	
6 USER PAGE
	6.1 Invite Code Card	邀请码介绍页面，随机生成邀请码，复制后分享给朋友用于注册SVIP优惠。邀请码分类：注册用户的邀请码奖励是使用者可以优惠5元；SVIP用户的邀请码奖励除了使用者有5元优惠，自己也可以有1周的SVIP延期；分销商邀请码的奖励是每个新注册SVIP用户返佣金20元现金，累积100元后1周结算。以上只有使用邀请码用户成为SVIP用户之后，奖励功能才能生效。
	6.2 user avatar和user name是注册者信息
	6.3 FAQ grade和FAQ  Score是趣味问答积分，答对一道题奖励10分，分数未来可以兑换奖品（奖品待定）
	6.4 collect section 用户艺术作品收藏列表，delete collect works可以删除收藏
	
7 AD_PAGE1
	是用户点击免费观看的作品，弹出说明页，告知用户需要观看广告才能体验的艺术作品。

8 AD_page2
	是微信激励广告页面，观看完广告后返回功能。
